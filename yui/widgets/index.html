YUI Widgets

  • AutoComplete

    Provides automatic input completion or suggestions for text input fields and textareas.
  • Button

    YUI's Button component is a mixed-module approach to providing a simple-to-use button that normalizes and enhances the Web browser's default button component.
  • Calendar

    Provides an interactive visual representation of a range of dates in one or more month grids
  • Charts

    The charts module provides a JavaScript API for visualizing data in a variety of formats accross all A-grade browsers.
  • DataTable

    The DataTable widget displays screen-reader accessible tabular data on a web page. Class extensions add functionality such as sorting and scrolling.
  • Dial

    The Dial widget is a circular value input control.
  • MenuNav Node Plugin

    Easily transform a list into an accessible and customizable drop-down menu.
  • Overlay

    Overlay is a positionable and stackable widget, which also provides support for the standard module format layout, with a header, body and footer section.
  • Panel

    The Panel is a UI Control that is meant to behave similarly to an OS window. The Panel control extends the functionality of Overlay, adding support for modality, close/dismiss buttons, autohiding and autofocus. Plugins to make it draggable and resizable are supported as well. Panel includes a pre-defined stylesheet to support default look and feel characteristics.
  • ScrollView

    ScrollView provides a scrollable content widget with flick velocity based inertia for both touch and mouse.
  • Slider

    The Slider widget is a UI control that enables the user to adjust values in a finite range along a horizontal or vertical axis.
  • TabView

    The TabView widget is a UI control that enables the user to switch between content panels.
