 Moving to 'gallery-xarno-plugin-collapse' and Y.Xarno.Plugin.Collapse

This simple, yet powerful, plugin gives you the ability to collapse and expand a node.

Configurations


  • lockWidth - allows (false) or disallows (true) the width to animate
  • lockHeight - allows (false) or disallows (true) the height to animate
  • state - setting the state to closed in the config will start the node as closed
  • easing - set the animation easing
  • duration - set the animation duration
  • endOverflow - if you need your overflow to be something other than hidden after the animation is complete, set it here

This sample will plug the div then animate it closed. After 4 seconds (3.5 after the default .5 duration passes) the div will animate open again.

Code Sample

<script src="http://yui.yahooapis.com/3.7.3/build/yui/yui-min.js"></script>
      <div id="collapsable" style="border:1px solid red;">
          <h1>Collapse Plugin Demo</h1>
          <p>Nunc a urna sed nibh semper sollicitudin in ut risumi tincidunt nec sodales libero condimentum.</p>
      </div>
 
 
        <script>
 
          YUI({
    //Last Gallery Build of this module
    gallery: 'gallery-2010.09.08-19-45'
}).use('gallery-plugin-collapse', 'node', function(Y){
            var n = Y.one('#collapsable');
            n.plug(Y.Plugin.Collapse);
            n.collapse.close();
            Y.later(4000, n, function(){ n.collapse.open();});
          });
 
        </script>
