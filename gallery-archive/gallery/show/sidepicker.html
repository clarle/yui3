Sidepicker (gallery-sidepicker)

Last Updated: 05/20/10
+ 1 -

This component is a "vertical" tab component.
It looks like a sidebar with several panel options to choose for. An animation shows the contents as opening when we want to review the actual contents.

Code Sample

<ul id="sidepicker">
	<li><a href="#first">First Tab</a></li>
	<li><a href="#second">Second Tab</a></li>
	<li><a href="#third">Third Tab</a></li>
</ul>
<div>
	<div id="first">
		Contents inside the first part. 
	</div>
	<div id="second">
		Contents inside the second part.
	</div>
	<div id="third">
		Contents inside the third part.
	</div>
</div>
 
<script>
YUI().use('gallery-sidepicker', function(Y) {
	var sidepicker = new Y.Sidepicker('sidepicker');
});
</script>
