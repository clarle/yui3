charts-legend (gallery-charts-legend)

Last Updated: 11/19/12

Jeff Craig

YUI Contributor

See 8 more by this user. Created: 01/28/11
Project: YUI 3
License: YUI BSD
YUI Version: 3.8.0pr1
Free for use.

This plugin must be attached to an instance of Y.Chart. Currently, it requires a single configuration option: container, which much be a Node or a Selector pointing to the element that should contain the legend. Future revisions will automatically create the container as the next sibling to the Chart container.

Code Sample

YUI().use('gallery-charts-legend', function(Y) {
 
    var myDataValues = [ 
        { category:"5/1/2010", values:2000, expenses:3700, revenue:2200 }, 
        { category:"5/2/2010", values:50, expenses:9100, revenue:100 }, 
        { category:"5/3/2010", values:400, expenses:1100, revenue:1500 }, 
        { category:"5/4/2010", values:200, expenses:1900, revenue:2800 }, 
        { category:"5/5/2010", values:5000, expenses:5000, revenue:2650 }
    ];
    var mychart = new Y.Chart({ width:400, height:300, dataProvider:myDataValues });
    mychart.plug(Y.Chart.Legend, { container: '#mylegend' });
    mychart.render("#mychart");
 
});
