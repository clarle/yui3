 Moving to 'gallery-xarno-list-expander' and Y.Xarno.Plugin.ListExpander

A light weight list manager that will add and remove an "open" className on the targeted <LI> with an optional param for closing the sibling <LI>s

Configuration


  • closeSiblings - (true) Specifies that the siblings, and sibling children should "close" when opening another <LI>

Code Sample

<script src="http://yui.yahooapis.com/3.7.3/build/yui/yui-min.js"></script>
<ul>
    <li>Option 1
        <ul>
            <li>Option 1 A</li>
        </ul>
    </li>
    <li>Option 2
        <ul>
            <li>Option 2 A</li>
        </ul>
    </li>
</ul>
 
YUI({
    //Last Gallery Build of this module
    gallery: 'gallery-2010.09.08-19-45'
}).use('gallery-list-expander', function(Y) {
 
    Y.one('ul').plug(Y.Plugin.ListExpander);
 
});
