TapToTop is just what the name implies: a button that smoothly scrolls the window back to the top. To use it, just plug it into the body of your page as shown in the code sample.

TapToTop is very easy to use. Just plug it into document.body. You can skin it however you like. A sample skin is provided in the Assets folder.

Code Sample

<script src="http://yui.yahooapis.com/3.4.1/build/yui/yui-min.js"></script>
YUI({
    //Last Gallery Build of this module
    gallery: 'gallery-2012.10.24-20-01'
}).use('gallery-taptotop', function(Y) {
 
    Y.one('body').plug(Y.TapToTopPlugin);    
 
});
