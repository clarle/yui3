joup (gallery-joup)

Last Updated: 10/18/11
+ 0 -

John George Wright

Created: 10/18/11
Project: YUI 3
License: YUI BSD
YUI Version: 3.4.1
Free for use.

The Joup img plugin creates a constrained hover status for img tags.

To use the Joup plugin, simply pass the reference to the plugin, a "altImage" parameter and any other optional parameters to the node's plug method.



<script>

  YUI().use("gallery-joup", function(Y)

  {

    Y.one("img#joup").plug(Y.Joup, {

      altImage: "/images/large-image.jpg"

    });

  });

<script>

The Joup plugin has several configuration properties that can be set via an object literal that is passed as a second argument to a Node instance's <code>plug</code> method.



<script>

  YUI().use("gallery-joup", function(Y)

  {

    Y.one("img#joup").plug(Y.Joup, {

      altImage: "/images/large-image.jpg",

      offsetLeft: 10,

      offsetRight: 40,

    });

  });

<script>
