Maybe the following example can help you:

http://developer.yahoo.com/yui/examples ... tamod.html

It seems deleteRow() accepts the row index you are going to delete:

myDataTable.deleteRow(index);

While deleteRows() accepts the starting row and total number of rows you are going to delete:

myDataTable.deleteRows(index, count);

If you want deleting all rows, borrowing your example, you can use (I didn't test it):

myDataTable.deleteRows(0, len);
