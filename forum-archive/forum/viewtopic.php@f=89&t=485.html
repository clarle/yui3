Panels & Dialogs: Make Resizable

Posted: Fri Jul 10, 2009 11:58 am
+0-
I wrote a general function to make panels and dialogs resizable. I generalized the YUI example on resizable panels.

Works well. Exception is that performance on IE 7 is barely OK.

Everything is generic for dialogs. For panels, you need to set one of the css rules for your panel. See the last css rule below.

Enjoy,

Larry

Code:
1) Add to your app's CSS:

/* resizable panel support. See the YUI example */
    .resizablepanel_c .bd {
        overflow:auto;
        background-color:#fff;
        padding:10px;
    }

    .resizablepanel_c .yui-resize-handle-br {
        right:0;
        bottom:0;
        height: 8px;
        width: 8px;
        position:absolute;
    }

    .resizablepanel_c.hide-scrollbars .yui-resize .bd {
        overflow: hidden;
    }

    .resizablepanel_c.show-scrollbars .yui-resize .bd {
        overflow: auto;
    }

    .resizablepanel_c.show-scrollbars .underlay {
        overflow: visible;
    }

/* This rule is NOT generic, you need to add selectors for any of your resizable panels.
    You do NOT need this rule for Dialogs
*/
    #a_resizable_panel .ft { 
        height:15px;
        padding:0;
    }

2. The function:
make_resizable = function(panel) {
  // Makes various changes to the panel or dialog's structure to support resizing
  // Call after the RENDER of the panel or dialog
  //
  // If resizing a panel: you need to create a min height for the ft.
  // No changes needed for dialogs since they always place buttons in the ft.
 
  // Add class resizablepanel_c to the container div
  var D = YAHOO.util.Dom;
  D.addClass(panel.element, 'resizablepanel_c');
 
  // locate the panel's main div and add resize support
  var el = D.getChildren(panel.element)[0],
      resize = new YAHOO.util.Resize(el, {
        handles: ['br'],
        autoRatio: false,
        minWidth: 300,
        minHeight: 100,
        status: false
      });

  // Add listener for re-sizes
  resize.on('resize', function(args) {
     var panelHeight = args.height;
     this.cfg.setProperty("height", panelHeight + "px");
     }, panel, true);
     
  // Add a constraint resize listener if the panel is constrained
  if (panel.cfg.getProperty("constraintoviewport")) {
    // Setup startResize handler, to constrain the resize width/height
    resize.on('startResize',
      function(args, params) {
        var D = YAHOO.util.Dom,
            vp_offset = YAHOO.widget.Overlay.VIEWPORT_OFFSET,
            resize = params,
            clientRegion = D.getClientRegion(),
            elRegion = D.getRegion(this.element);
        resize.set("maxWidth", clientRegion.right - elRegion.left - vp_offset);
        resize.set("maxHeight", clientRegion.bottom - elRegion.top - vp_offset);
      }, resize, panel);
  }
}; 


Usage example
Code:
    var dialog = new YAHOO.widget.Dialog(chooser_id, {
                    width : "40em",
              fixedcenter : true,
                  visible : false,
                     modal: true,
                postmethod: "form",
      constraintoviewport : true,
                  buttons : [{ text:"Cancel", handler: function(){this.cancel();} }]
         });

.... // set initial Dialog body, header, etc

    dialog.render(document.body);
    make_resizable(dialog);
